Search

  1. Location(Auto): This input would give a filtered results which takes this location as it's center
  2. Hospital Name: This input would give Hospitals with name which contains the input
  3. Rating Filter: This filter would give hospitals with ratings higher than the inputted rating
  4. Availability Filter: This filter would give hospitals with availability of the specified facility that is higher than the inputted percentage
  5. Cost Filter: This filter would give hospitals with their average cost being in between the costs you input
